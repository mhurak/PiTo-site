PiTo · iOS aplikacia pre hydrataciu

Pijes lepsie, citis sa lepsie.

PiTo ti pomaha drzat pitny rezim bez zbytocnej zlozitosti. Rychle pridanie vody, prehladny denny stav, statistiky, widget a Live Activity priamo na lock screene.

GitHub stranka Kod aplikacie

Co PiTo vie dnes

Denny prehlad

Aktualny progres v percentach, rychle pridanie (200/300/500 ml) aj manualny vstup vlastneho objemu.

Pokrocile statistiky

Streak, uspesnost 7/30 dni, trend oproti minulemu tyzdnu, hodinovy profil a najdlhsia pauza medzi pitiami.

Widget + Live Activity

Widget s rychlym pridanim 200/300 ml a Live Activity v Dynamic Islande/Lock screene pre okamzity prehlad.

Praca s historiou

Swipe mazanie s obnovenim, vizualne rozlisenie odstranenej polozky a korektny prepocet denneho objemu.

Kratky status projektu

iOSSwiftUI + SwiftData
HealthKitImport onboarding dat
WidgetKitRychle pridanie vody
LiveDynamic Island podpora
